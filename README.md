### Key Etymology (Ithkuil I - Iţkuîl)

From root **-K-L-** (speech/voice/interpretation):
- `kul` = "a word" (Stem 2 Holistic)
- `kuîl` = "an imaginary word" (Secondary Mode)
- `ţkuîl` = "a vocabulary of imaginary words" (Composite Configuration)
- `iţkuîl` = "a complete imaginary language" (Delimitive + Coalescent)

**Final meaning**: "an idea/fantasy of a complete purposeful system of complementary speech elements" = **"an imaginary language"**
