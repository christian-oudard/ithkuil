# Chapter 2: Morpho-Phonology

**A GRAMMAR OF NEW ITHKUIL - A CONSTRUCTED LANGUAGE**

## 2.0 Introduction

Morpho-phonology refers to how a language uses its phonemes (meaningful sounds) and phonological features (e.g., syllabic stress, gemination, tone, etc.) to generate patterns for word-formation and morphological categories.

### Parts of Speech

New Ithkuil has three parts of speech:
1. **Formatives** - Nouns and verbs combined into a single part of speech
2. **Adjuncts** - Modifiers (like English adverbs, modals)
3. **Referentials** - Pronouns

### Grammatical Typology

New Ithkuil is an agglutinative language with a slot-based word structure.

## 2.1 The Standard Vowel-Form Sequence

| | Series 1 | Series 2 | Series 3* | Series 4 |
|---|---|---|---|---|
| **Form 1** | a | ä | ia / uä | ai |
| **Form 2** | e | i | ie / uë | au |
| **Form 3** | u | ü | io / üä | ei |
| **Form 4** | o | ö | iö / üë | eu |
| **Form 5** | ë | ê | eë / öë | ëi |
| **Form 6** | î | û | uö / öë | ëu |
| **Form 7** | â | ô | uo / öä | oi |
| **Form 8** | ê | î | ue / ië | ou |
| **Form 9** | ô | û | ua / iä | ui |

*When preceded by **y**-, Series 3 forms beginning with -**i** use their alternate forms (e.g., **yuä** not **yia**). Series 3 forms beginning with -**u** use alternate forms if preceded by **w**- (e.g., **wiä** not **wua**).

## 2.2 Rules for Inserting a Glottal-Stop

When a glottal-stop must be infixed into a vowel-form **V**:

1. If **V** is a single vowel or diphthong, place the glottal-stop after **V** (e.g., **a** → **a'**, **ai** → **ai'**)
2. If **V** is a disyllabic conjunct, place the infix between the two syllables (e.g., **ua** → **u'a**)
3. If Rule 1 creates a phonotactically impermissible conjunct:
   - If **V** is a single vowel, reduplicate it after the glottal-stop (e.g., **a** → **a'a**)
   - If **V** is a diphthong, place the glottal-stop between the two vowels (e.g., **ai** → **a'i**)

## 2.3 Formative Slot Structure

A formative has 10 slots:

| Slot | Name | Content |
|------|------|---------|
| I | CC | Concatenation status (glottal stop, h-, hw-, w-, y-) |
| II | VV | Version + Stem (or Slot IV/VI shortcut) |
| III | CR | Root (1-5 consonants) |
| IV | VR | Function + Specification + Context |
| V | (CSVX...) | Affixes applying to stem only |
| VI | CA | Configuration + Affiliation + Perspective + Extension + Essence |
| VII | (VXCS...) | Affixes applying to stem + CA |
| VIII | (VN CN) | Valence + Mood/Case-Scope (or Aspect/Phase/Level/Effect) |
| IX | VC/VK | Case (or Format, or Illocution + Validation) |
| X | [stress] | Syllabic stress determines Slot IX interpretation |

### Stress Patterns (Slot X)

- **Penultimate stress**: Slot IX = Case, unframed Relation
- **Ultimate stress**: Slot IX = Case, unframed Relation (alternate)
- **Antepenultimate stress**: Slot IX = Case, framed Relation

## 2.4 Roots, Stems, and Specifications

### 2.4.1 Roots

The **root** is the semantic basis for noun/verb stems. It consists of a consonantal form **CR** in Slot III (1-5 consonants). The phonotactic rules allow for over 33,000 possible roots.

Example: **-DN-** means 'NAME/DESIGNATION/LABEL'

### 2.4.2 Formatives as Nouns and Verbs

All formatives can function as both nouns and verbs. **-DN-** means both 'a name' and 'to name'. The distinction is made by analyzing morpho-phonological structure and morpho-syntactic relationships.

### 2.4.3 Stems

Each root has three stems plus "Stem Zero":

- **Stem 1** (Slot II: **a**): e.g., **-adn-** = "(to be) a name"
- **Stem 2** (Slot II: **e**): e.g., **-edn-** = "(to be) a designation"
- **Stem 3** (Slot II: **u**): e.g., **-udn-** = "(to be) a label"
- **Stem 0** (Slot II: **o**): amalgamation of all three stem meanings

### 2.4.4 Specification

Four specifications distinguish the semantic focus:

| Specification | Abbr. | Vowel | Description |
|---------------|-------|-------|-------------|
| **BASIC** | BSC | a | Holistic combination of CTE and CSV |
| **CONTENTIAL** | CTE | ä | The content, essence, or meaningful aspect |
| **CONSTITUTIVE** | CSV | e | The form, structure, or physical makeup |
| **OBJECTIVE** | OBJ | i | The object/instrument involved in an act |

## 2.7 Pitch Accent Rules

The language employs a pitch accent system for parsing word boundaries:

1. Normal sentences use default pitch
2. Yes/no questions may use RISING pitch on the last word
3. Non-narrative Register clauses may use LOW tone on first and last words
4. Rules for monosyllabic words and word-initial stress:
   - After FALLING/RISING-FALLING/LOW pitch → use FALLING/FALLING-RISING/HIGH
   - After HIGH/RISING/FALLING-RISING pitch → use RISING/RISING-FALLING/LOW
5. Special parsing adjunct **'V'** (vowel between two glottal stops) can mark word boundaries:
   - **'a'** = penultimate stress
   - **'e'** = ultimate stress
   - **'o'** = antepenultimate stress
   - **'u'** = preantepenultimate/initial stress
