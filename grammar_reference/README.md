## Quick Reference: Formative Slot Structure

```
Slot I   (CC)   - Concatenation status (', h-, hw-, w-, y-)
Slot II  (VV)   - Version + Stem (or IV/VI shortcut)
Slot III (CR)   - Root (1-5 consonants)
Slot IV  (VR)   - Function + Specification + Context
Slot V   (CSVX) - Affixes applying to stem only
Slot VI  (CA)   - Configuration + Affiliation + Perspective + Extension + Essence
Slot VII (VXCS) - Affixes applying to stem + CA
Slot VIII(VNCN) - Valence + Mood/Case-Scope (or Aspect/Phase/Level/Effect)
Slot IX  (VC/VK)- Case (or Format, or Illocution + Validation)
Slot X   [stress] - Determines Slot IX interpretation
```

## Key Morphological Categories

### Slot VI: CA Complex (agglutinated)
- **Configuration** (20): UPX, DPX, MSS, MSC, MSF, MDS, MDC, MDF, MFS, MFC, MFF, DSS, DSC, DSF, DDS, DDC, DDF, DFS, DFC, DFF
- **Affiliation** (4): CSL, ASO, COA, VAR
- **Perspective** (4): M, G, N, A
- **Extension** (6): DEL, PRX, ICP, ATV, GRA, DPL
- **Essence** (2): NRM, RPV

### Slot IX: Cases (68 in 8 groups)
1. Transrelative (9): THM, INS, ABS, AFF, STM, EFF, ERG, DAT, IND
2. Appositive (9): POS, PRP, GEN, ATT, PDC, ITP, OGN, IDP, PAR
3. Associative (9): APL, PUR, TRA, DFR, CRS, TSP, CMM, CMP, CSD
4. Adverbial (9): FUN, TFM, CLA, RSL, CSM, CON, AVR, CVS, SIT
5. Relational (8): PRN, DSP, COR, CPS, COM, UTL, PRD, RLT
6. Affinitive (8): ACT, ASI, ESS, TRM, SEL, CFM, DEP, VOC
7. Spatio-Temporal I (8): LOC, ATD, ALL, ABL, ORI, IRL, INV, NAV
8. Spatio-Temporal II (8): CNR, ASS, PER, PRO, PCV, PCR, ELP, PLM
