# Chapter 3: Basic Morphology

**A GRAMMAR OF NEW ITHKUIL - A CONSTRUCTED LANGUAGE**

## 3.0 Introduction

This chapter covers mandatory morphological categories applying to both nominal and verbal formatives:
- **Configuration**, **Affiliation**, **Perspective**, **Extension**, **Essence** (Slot VI)
- **Version** (Slot II)
- **Function**, **Context** (Slot IV)

## 3.1 Configuration

Configuration deals with physical similarity and relationships between members of a noun referent within groups, collections, sets, etc.

### Three Factors of Configuration

1. **Plexity** - three-way distinction:
   - **Uniplex (U)** - single or internally unified
   - **Duplex (D)** - binary or two-halved
   - **Multiplex (M)** - three or more components

2. **Separability** - three-way distinction (for non-uniplex):
   - **Separate (S)** - physically separate
   - **Connected (C)** - connected in some way
   - **Fused (F)** - fused together

3. **Similarity** - three-way distinction (for non-uniplex):
   - **Similar (S)** - physically similar
   - **Dissimilar (D)** - physically dissimilar
   - **Fuzzy (F)** - similarity is subjective/unascertainable

### The 20 Configurations

| Abbr | Meaning | Abbr | Meaning |
|------|---------|------|---------|
| UPX | uniplex | DPX | duplex |
| MSS | multiplex/similar/separate | DSS | duplex/similar/separate |
| MSC | multiplex/similar/connected | DSC | duplex/similar/connected |
| MSF | multiplex/similar/fused | DSF | duplex/similar/fused |
| MDS | multiplex/dissimilar/separate | DDS | duplex/dissimilar/separate |
| MDC | multiplex/dissimilar/connected | DDC | duplex/dissimilar/connected |
| MDF | multiplex/dissimilar/fused | DDF | duplex/dissimilar/fused |
| MFS | multiplex/fuzzy/separate | DFS | duplex/fuzzy/separate |
| MFC | multiplex/fuzzy/connected | DFC | duplex/fuzzy/connected |
| MFF | multiplex/fuzzy/fused | DFF | duplex/fuzzy/fused |

## 3.2 Affiliation

Affiliation distinguishes member relationships in terms of subjective purpose, function, or benefit.

### The 4 Affiliations

| Affiliation | Abbr | Affix | Description |
|-------------|------|-------|-------------|
| **Consolidative** | CSL | (null) | Natural occurrence, no shared function |
| **Associative** | ASO | -l- | Members share same function/purpose |
| **Coalescent** | COA | -r- | Complementary functions serving greater role |
| **Variative** | VAR | -ř- | Different, unrelated functions/purposes |

**Examples:**
- grove (CSL) vs. orchard (ASO)
- toolset (COA) - each tool distinct but complementary
- jumble of tools (VAR) - random, unrelated purposes

## 3.3 Perspective

Perspective is the closest equivalent to Number in other languages.

### The 4 Perspectives

| Perspective | Abbr | Affix | Description |
|-------------|------|-------|-------------|
| **Monadic** | M | (null) / -l- | Single embodiment of configuration |
| **Agglomerative** | G | -r- | Fuzzy "non-count" number |
| **Nomic** | N | -w- / -v- | Generic collective/archetype |
| **Abstract** | A | -y- / -j- | Abstract concept, "X-hood" |

**Singulative Equivalent:** Non-count nouns use monadic for smallest salient manifestation (e.g., 'a drop of water', 'a single hair').

## 3.4 Extension

Extension indicates how a noun or verb is considered in terms of spatial/temporal extent.

### The 6 Extensions

| Extension | Abbr | Affix | Description |
|-----------|------|-------|-------------|
| **Delimitive** | DEL | (null) | Contextual entirety, clear boundaries |
| **Proximal** | PRX | -t- / -d- | Portion relevant to context |
| **Inceptive** | ICP | -k- / -g- | Beginning, initiation |
| **Attenuative** | ATV | -p- / -b- | End, termination |
| **Graduative** | GRA | -g- / -gz- | Gradual onset, "fade-in" |
| **Depletive** | DPL | -b- / -bz- | Gradual terminus, "fade-out" |

## 3.5 Essence

Essence distinguishes real-world actualities from alternative/imagined counterparts.

### The 2 Essences

| Essence | Abbr | Description |
|---------|------|-------------|
| **Normal** | NRM | Real-world actualities (default) |
| **Representative** | RPV | Alternative, imagined, potential counterparts |

**RPV affixes modify Perspective:**
- Monadic: null → -l- (or -tļ- if standalone)
- Agglomerative: -r- → -ř-
- Nomic: -w- → -m- (or -h- after Ct, Cp, Ck)
- Abstract: -y- → -n- (or -ç- after Ct, Cp, Ck)

## 3.6 The CA Affix-Complex

Slot VI contains an agglutinated consonant complex encoding all five categories:

**Order: Affiliation + Configuration + Extension + Perspective + Essence**

Default form (CSL/UPX/DEL/M/NRM): **-l-**

### Gemination Rules (when Slot V contains affixes)

1. Single consonants: geminate (e.g., **p → pp**)
2. Stop + liquid/approximant: geminate the stop
3. Forms with sibilant: geminate the sibilant
4. Non-sibilant fricative/nasal initial: geminate it
5. Voiceless stop + fricative: geminate the fricative

## 3.7 Version

Version indicates telicity - whether an entity/act is goal-oriented.

### The 2 Versions

| Version | Abbr | Slot II | Description |
|---------|------|---------|-------------|
| **Processual** | PRC | a/e/u/o | Default, process-oriented |
| **Completive** | CPT | ä/i/ü/ö | Goal/result-oriented |

**Examples:**
- to hunt (PRC) → to hunt down (CPT)
- to study (PRC) → to learn (CPT)
- to search for (PRC) → to find (CPT)

## 3.8 Function

Function distinguishes static states from dynamic actions.

### The 2 Functions

| Function | Abbr | Description |
|----------|------|-------------|
| **Stative** | STA | Static, unchanging state/entity |
| **Dynamic** | DYN | Action, change, movement |

Function is shown in the Slot IV vowel.

## 3.9 Context

Context indicates which features of a formative are being psychologically implied.

### The 4 Contexts

| Context | Abbr | Description |
|---------|------|-------------|
| **Existential** | EXS | Ontologically objective features, mere identification |
| **Functional** | FNC | Socially defined features: purpose, function, benefit |
| **Representational** | RPS | Symbolic, metaphorical, metonymic use |
| **Amalgamative** | AMG | Systemic, holistic, gestalt-like consideration |

### Slot IV VR Values

| Function | Spec | EXS | FNC | RPS* | AMG |
|----------|------|-----|-----|------|-----|
| **STA** | BSC | a | ai | ia/uä | ao |
| | CTE | ä | au | ie/uë | aö |
| | CSV | e | ei | io/üä | eo |
| | OBJ | i | eu | iö/üë | eö |
| **DYN** | BSC | u | ui | ua/iä | oa |
| | CTE | ü | iu | ue/ië | öa |
| | CSV | o | oi | uo/öä | oe |
| | OBJ | ö | ou | uö/öë | öe |

*RPS forms use alternates when preceded by -y- or -w-

## 3.10 Slot I/II Shortcuts

When Slot I contains **w-** or **y-**, Slots IV and VI are elided, and Slot II conveys:
- STA Function, BSC Specification, EXS Context
- Plus specific CA values as determined by the shortcut table

This allows shorter words when default values apply.
